Hello, and welcome to the home page of Introduction to JavaScript (IAP 2014)!
Prepare to be amazed by the wonders of JS on both the client and the server as you complete a fast-paced adventure through the workings and use of the language. In particular, we'll be covering:

  • DOM (HTML element) manipulation
  • JavaScript libraries - lots of jQuery, mention of AngularJS, D3, 3SSC, and others
  • Console debugging and developer tools
  • AJAX/Asynchronous HTTP requests
  • JavaScript on the server using Node.js and Express

Classes are on Tuesdays and Thursdays 1/9 to 1/30 from 2:00 to 4:00 pm.
The first hour will be consist of lecture and activities. The second hour will be used for pair-debugging, during which students work in pairs (or small groups) to debug their code and solidify their understanding of the material.
