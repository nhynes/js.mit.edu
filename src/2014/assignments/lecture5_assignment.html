Introduction to JavaScript - Problem Set 5

If you haven't already done, so, you'll want to install the Node.js web server application. Navigate to nodejs.org and click on the install button. This will download an installer for your platform or the binaries if you're using Linux.

  • Linux - move the binary files into the correct locations (the folder names give you this info)
  • Macintosh & Windows - run the installer

Warm Up

    [Node.js]
  1. Verify that your install worked by making your own "Hello world" server!
    Open the command line and use the 
    cd
    command to navigate to your server's JS file and execute 
    node yourfile.js
    . Your server should start serving your website at localhost:yourPortNumber
