Introduction to JavaScript - Problem Set 2

If you've already seen the first assignment, then welcome back! Otherwise, you seem to have stumbled upon the second assignment of Introduction to JavaScript!

For this assignment and the rest of your JavaScript programming career (which will be forever, right? (JavaScript, not the assignment...)), you'll need a text editor suitable for programming. If you already have one, then you're already good to go. If not, I would recommend Brackets or Sublime Text. Brackets is nice because it's designed specifically for web development and, along with having some cool web dev features like live-preview and inline CSS editing, it allows you to write editor extensions in HTML, JS, and CSS (and it's free!). Sublime Text is a general-purpose text editor that can be used for any language or build system; unfortunately, it's not free, but it does have an unlimited trial period.

Here's a refresher on how to open the console, in case you forgot:

  • If you are using Firefox: right click on this page, select "Inspect Element," and then navigate to the "Console" tab (on the left)
  • If you are using Chrome: right click on this page, select "Inspect Element," and then navigate to the "Console" tab (on the right)
  • If you are using Internet Explorer: click on the start menu, select "Shut down," and install a better operating system try to use a more developer-friendly browser

We'll be using another useful tool that comes with your browser: the Inspector. While not a detective, the Inspector allows you to examine and modify HTML elements and their properties. This is the dialog that pops up when you select "Inspect Element".
