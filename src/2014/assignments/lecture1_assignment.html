Hi! Congratulations on making it to the first assignment of Introduction to JavaScript. If you're wondering how to actually get your code to run, DON'T PANIC. Here's a quick tutorial:

  • If you are using Firefox: right click on this page, select "Inspect Element," and then navigate to the "Console" tab (on the left)
  • If you are using Chrome: right click on this page, select "Inspect Element," and then navigate to the "Console" tab (on the right)
  • If you are using Internet Explorer: right click on the window, select "Close," and promptly leave the class try to use a more developer-friendly browser

From the console, you can enter commands that will be executed by the JavaScript interpreter. If you get stuck, try typing in an object followed by a dot to see its properties, and then pick one that looks like it does what you want to do. Also, any console.log()'s that you run will also be printed here. Go ahead, try it out!

I realize that this is a bit tedious. If you'd like to use a more "full-featured" "editor," you can try JSFiddle or just copy and paste in from your favorite text editor. Fortunately, we'll be covering loading JS from files in the next lecture.

Also, the height of the text box indicates the approximate number of lines it should take you to answer the question. Don't worry about conciseness, though; focus on correctness.

    2. ["Classes"]
  2. Create a class named 
    Person
    that has the following properties: 
    • firstName
      - the Person's first name
    • lastName
      - the Person's last name
    • friends
      - a list of Person objects that are friends with this Person (each friend also has Person as a friend)
    • getFullName()
      - returns the Person's full name (first name *space* last name)
    • friend(newFriend)
      - makes the Persons friends (adds each other to their friend lists) if they previously were not
    • defriend(noLongerFriend)
      - makes the Persons not friends (removes each other from their friend lists) if they were previously (hint: use the splice() function of the Array object)
    • isFriendsWith(person)
      - returns true if the Persons are friends, false otherwise (hint: use the indexOf() function of the Array object)
    • toString()
      - returns a String representing the Person in the following format: first name *space* last name *space* has N friends, where N is the number of friends the Person has. If N is equal to 1, 'friends' should become 'friend'
