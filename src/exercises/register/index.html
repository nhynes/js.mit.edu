<blurb>
We'll be using a technique known as "active learning." Woah, woah, hold up, don't run away yet. If you're skeptical (or traumatized by TEAL), here's the rationale:

  1. Learning a new programming language requires programming in that language[citation needed].
  2. There are already plenty of static or semi-interactive tutorials on JS.
Instead of wasting your time regurgitating (perhaps pre-chewed) tutorials and documentation, I'm going to focus on designing high-quality examples and exercises. We'll work on these during class so that you can take advantage of the full interactivity of a real-life course. However, it'll help if you acquaint yourself with topics covered in each class before class. Reading tutorials is boring, so only tasteful links to external resources will be provided for your preparatory pleasure.
</blurb>

So, without further ado, I give you... the suggested readings!
