Practice problems

    [Forms]
  1. Create a form that contains a text input field for Name and House and a submit button. Set the 
    id
    attributes of the inputs to "name" and "house".
    Name should only contain the characters A-Z and spaces and House should be either Gryffindor, Hufflepuff, Ravenclaw, or Slytherin. The submit button should be titled "Enter Chamber of Secrets". Although you can use the 
    pattern
    attribute for validation, you should use JavaScript for this question.
    If the input is incorrect, a span of red-colored text should appear at the bottom of the form, displaying the text "Try again, mortal." If the input is correct, a span of green-colored "In exchange for your soul, you may enter the Chamber." should appear below the form.
    Don't forget to import the HTML checker (http://introjsiap.com/js/htmlchecker.js) after jQuery and set the title of the page to Assignment 3 - Practice problem #1!
    2. [HashChangeEvent, "Classes"]
  2. Create a "social networking site" for Person objects. The Person's page should be located at thefile.html#lowercasefullnamewithoutspaces and should contain a top-level heading displaying the Person's full name, and a list of the Person's friends' names that link (think 
    <a>
    ) to the friends' pages.
    You'll want to import jQuery and the HTML checker (don't forget to set the title of the page to Assignment 3 - Practice problem #2!). The HTML checker will provide the function 
    getPeople()
    which will return a list of Person objects for you to use.

    transform
